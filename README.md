# Presence-Voice-Tracker-Bot
A Discord bot tracking user presences and voice sessions, generating HTML and CSV reports, handling user consent, and managing data securely. Ένα Discord bot για καταγραφή παρουσιών και φωνητικών συνεδριών, δημιουργία αναφορών σε HTML και CSV, με έλεγχο συγκατάθεσης χρηστών και διαχείριση δεδομένων με ασφάλεια.
